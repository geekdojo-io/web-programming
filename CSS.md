# CSS - Cascading Style Sheets

### Separate Style

HTML
```html
<h1>Hello World</h1>
<h1>James Bond</h1>
<h2>Secret Agent</h2>
<h3>Big Bob</h3>
```

CSS
```css
h1 {color: blue;}
```
